# 🧙‍♂️ Persona Weaver (User Trình tạo nhân vật)

[Tiếng Trung] | [English](#english)

SillyTavern Tiện ích mở rộng gốc. Nhằm mục đích sử dụng AI tạo thông minh, trau chuốt chuyên sâu và quản lý User Persona(nhân vật người dùng), hỗ trợ chỉnh sửa so sánh thông minh và tự động liên kết với World Info.

## ✨ Tính năng cốt lõi

*   **AI Tạo thông minh**:Dựa trên mô tả ngôn ngữ tự nhiên đơn giản (hoặc tích hợp sẵn YAML mẫu), tự động tạo nhân vật người dùng có cấu trúc, chất lượng cao.
*   **Trau chuốt sâu & So sánh thông minh**:
    *   Không hài lòng với thiết lập hiện tại? Nhập ý kiến chỉnh sửa,AI viết lại thông minh.
    *   Cung cấp **Diff (So sánh khác biệt)** chế độ xem, hiển thị trực quan các thay đổi trước và sau khi sửa, hỗ trợ giữ lại có chọn lọc hoặc chỉnh sửa trực tiếp.
*   **Hệ thống hộp nháp**:Tự động hoặc thủ công lưu lịch sử tạo vào hộp nháp, không bao giờ mất cảm hứng, điền lại bằng một cú nhấp chuột bất cứ lúc nào.
*   **Liên kết World Info**:Tự động phát hiện World Info đang liên kết, ghi thiết lập thành mục nhập bằng một cú nhấp chuột, thực hiện liên kết cốt truyện sâu sắc.
*   **Độc lập API Hỗ trợ**:Hỗ trợ cấu hình độc lập API(như OpenAI,DeepSeek v.v.) để tạo trong nền, không chiếm ngữ cảnh và cấu hình mô hình của cuộc trò chuyện chính.
*   **Tương thích di động**:Đã tối ưu hóa UI thiết kế, hỗ trợ thao tác vuốt chạm và nổi trên điện thoại di động.

## 📦 Cách cài đặt

1.  **Yêu cầu trước**:Khuyên dùng cài đặt và bật [TavernHelper (JS-Slash-Runner)](https://github.com/n0vi028/JS-Slash-Runner) plugin để có trải nghiệm thao tác World Info tốt nhất (không bắt buộc, nhưng khuyên dùng).
2.  Mở SillyTavern của **Extensions (Tiện ích)** trang.
3.  Nhấp vào **Install Extension**.
4.  Tại URL nhập địa chỉ kho lưu trữ này:`[https://github.com/sisisisilviaxie-star/st-persona-weaver]`
5.  Nhấp vào **Save** và làm mới trang.

## 📖 Tóm tắt sử dụng

1.  Sau khi cài đặt, phía trên hộp nhập liệu sẽ xuất hiện **Biểu tượng đũa phép** (<i class="fa-solid fa-wand-magic-sparkles"></i>),nhấp để mở bảng điều khiển.
2.  **Tạo**:Nhập yêu cầu của bạn trên trang chỉnh sửa, nhấp vào tạo.
3.  **Trau chuốt**:Chọn một đoạn văn bản trong kết quả đã tạo, nhấp vào nút"Sửa"hiện ra, hoặc trực tiếp nhập ý kiến bên dưới để trau chuốt.
4.  **Lưu**:Nhấp vào"Lưu"Lưu kết quả vào bản nháp, hoặc nhấp vào"Ghi đè nhân vật"để áp dụng trực tiếp.

## English

**Persona Weaver** is a native extension for SillyTavern that uses AI to help create, refine, and manage User Personas, with automatic World Info synchronization.

### ✨ Features

*   **AI Generation**: Automatically generate detailed, structured User Personas (YAML) based on simple prompts or templates.
*   **Smart Refinement & Diff View**: 
    *   Refine your persona with natural language instructions.
    *   **Smart Contrast**: Visually compare the original vs. refined text side-by-side, allowing you to selectively apply changes or edit directly.
*   **Drafts System**: Auto-save your generation history to the Drafts tab. Never lose an idea again.
*   **World Info Sync**: Automatically detects bound World Info books and allows one-click saving of your persona as an entry.
*   **Independent API**: Supports configuring a separate API (e.g., OpenAI, DeepSeek) for generation tasks, keeping your main chat context free.
*   **Mobile Friendly**: Optimized UI for touch screens and mobile devices.

### 📦 Installation

1.  **Prerequisite**: [TavernHelper (JS-Slash-Runner)](https://github.com/n0vi028/JS-Slash-Runner) is recommended for full World Info features.
2.  Open **Extensions** in SillyTavern.
3.  Click **Install Extension**.
4.  Paste the repo URL: `https://github.com/sisisisilviaxie-star/st-persona-weaver`
5.  Click **Save** and reload.
